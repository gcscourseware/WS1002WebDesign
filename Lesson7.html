  • WordPress: SEO plugins (like Yoast) automatically generate and submit your sitemap to Google via Search Console.

These elements are what search engines and users see first in search results.


Video Demo of Setting URL Slug, Meta Tags and Description

The Snippet: Meta Title and Description

The **Meta Title** and **Meta Description** form the "snippet" displayed in Google search results.

  • Meta Title: Should be unique, include the page's main keyword, and be under 60 characters.
  • Meta Description: Should be a compelling, keyword-rich summary (under 160 characters) that encourages the user to click.
  • Location: These are set in the 'SEO' section of the individual Page Settings.

URLs and Alt Text:

  • Custom URLs (Slugs): Ensure your URL is clean, short, and keyword-rich (e.g., /best-coffee-beans instead of /page-id-23847).
  • Alt Text (Image Optimization): This text describes the image content. It improves accessibility and helps search engines index your images. Always include relevant keywords here.

Boosting Site Speed:

  1. Compress Images: Never upload huge, unoptimized files. Use tools to compress images (JPEG, WebP) before uploading.
  2. Lazy Loading: Ensure images and videos "lazy load," meaning they only load when the user scrolls down to view them, not immediately upon page load. (Most modern builders handle this automatically).
  3. Minimize Apps/Plugins: Every app/plugin adds code. Only install and keep the tools you actively use. Too many can severely slow down your site.
Test Your Speed: Regularly check your site's performance using Google's PageSpeed Insights tool. Aim for green scores for both mobile and desktop.

Mobile Responsiveness:

Always review and manually adjust your site's layout using the editor's dedicated mobile preview mode.

  • Ensure text is readable (no tiny fonts).
  • Buttons are large enough to tap (good touch targets).
  • Elements are not overlapping or hidden off-screen.
