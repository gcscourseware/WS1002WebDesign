Content builders make adding media straightforward. The key is optimization and placement.

Text Management:

Editing: In most builders (like Wix), you simply click the text box to edit content. Use the inline text editor (toolbar) to set headings (H1, H2, P), adjust alignment, and apply links.

  • SEO Tip: Only use one H1 tag per page. Use H2s and H3s for subheadings and structural hierarchy.

Image and Video Handling:

The Media Manager: This central library stores all uploaded assets. Always upload the highest quality image you have, and let the builder optimize the size for faster loading.

  • Image Best Practice: After uploading, always add Alt Text (Alternative Text). This describes the image for screen readers and search engines, improving accessibility and SEO.
  • Video Embedding: For videos, never upload the file directly unless absolutely necessary. Instead, use the video element to embed content hosted on YouTube or Vimeo. This prevents slow site loading.

üìÑ 2. Creating and Managing Pages

The "Pages" menu is where you define your site's architecture. A logical structure is essential for user experience.

Steps for Page Management:

  1. Adding a Page: Click the "+" icon or "Add Page" button in the Pages menu. You can start with a blank page, or choose from pre-designed layout options (e.g., Contact page, Services page).
  2. Naming: Give the page a clear, concise name (e.g., "Our Team," "Pricing"). This name often defaults to the menu label.
  3. Page Settings: Access settings to manage important metadata:
    • SEO: Set the Page Title and Description that appear in search results.
    • URL Slug: Customize the address (e.g., change `/our-story-long` to `/about`).
    • Permissions: Choose whether the page is public or restricted (e.g., members only).
  4. Duplicating & Hiding: If you need a new page with the same layout, duplicating an existing page is a massive time saver. You can also hide unfinished pages from the public until they are ready.

üó∫Ô∏è 3. Structuring Navigation and Menus

Navigation is the roadmap for your users. Good navigation should be intuitive, discoverable, and consistent.

Rule of Thumb: If a user can't find what they need in three clicks or less, your navigation is too complex.

Key Menu Elements:

Primary Menu (Header):

This is the main navigation bar at the top of your site. It should only contain links to your most critical pages (Home, About, Services/Products, Contact).

  • Best Practice: Keep it to 5-7 items maximum.
Creating Sub-Menus (Dropdowns):

To create a nested menu (e.g., Services > Web Design, Services > SEO), you typically drag the sub-page slightly underneath and to the right of its parent page within the Pages management panel. This creates the visual hierarchy.

Note: On mobile, these dropdowns usually collapse into an accordion style within the hamburger menu.

Footer Navigation:

Use the footer for secondary, less critical links: Privacy Policy, Terms of Use, Sitemap, and other informational pages that don't need prime real estate in the header.
